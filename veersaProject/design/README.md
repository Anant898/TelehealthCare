# Design Files

This directory should contain:

1. **Figma/Adobe XD Design Files**
   - Patient registration flow (web and mobile)
   - Payment screen (web and mobile)
   - Video consultation interface (web and mobile)
   - Chat interface (web and mobile)
   - Patient dashboard (web and mobile)

2. **Information Architecture**
   - See `../docs/InformationArchitecture.md` for user flows and navigation structure

3. **Design System**
   - See `../docs/DesignSystem.md` for color palette, typography, and UI components

## Design Theme

The application uses a consistent medical theme with:
- Primary color: Medical blue (#0066CC)
- Clean, professional appearance
- Healthcare-focused iconography
- Accessible color contrasts

## Note

Design mockups should be created using Figma, Adobe XD, or similar design tools and linked/uploaded to this repository or hosted online with links provided in the main README.
